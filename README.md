# Meridian — Small Business Website Demo

A polished, fully responsive demo website template designed to fit and appeal to **any small business** — cafés, salons, studios, boutiques, and service providers alike.

Built with plain **HTML, CSS, and JavaScript** — no build step, no dependencies. Just open `index.html`.

## ✨ Features on display

| Area | What it shows off |
|------|-------------------|
| **Light / dark mode** | One-click theme toggle that remembers your choice and respects system preference |
| **Hero** | Animated gradient blobs, floating "stats" card, and a live count-up metric |
| **Sticky nav** | Blur-glass header, scroll progress bar, active-section highlighting, mobile hamburger menu |
| **Services** | Six feature cards with animated gradient borders on hover |
| **About + stats** | Layered visual composition and count-up statistics triggered on scroll |
| **Work gallery** | Filterable portfolio grid (All / Food / Wellness / Retail / Services) |
| **Pricing** | Monthly ↔ yearly billing toggle with animated price changes and a highlighted plan |
| **Reviews** | Auto-playing testimonial slider with dots, arrows, and swipe support |
| **FAQ** | Smooth single-open accordion |
| **Contact** | Client-side form validation with inline errors + success state |
| **Extras** | Scroll-reveal animations, back-to-top button, newsletter signup, animated marquee |

Accessibility touches: skip link, ARIA roles/labels, keyboard-friendly controls, and `prefers-reduced-motion` support.

## 🗂 Structure

```
.
├── index.html        # All sections and markup
├── css/styles.css    # Design system, themes, layout, animations
└── js/main.js        # All interactivity (vanilla JS, no libraries)
```

## ▶️ Preview locally

```bash
# any static server works, for example:
python3 -m http.server 8000
# then visit http://localhost:8000
```
